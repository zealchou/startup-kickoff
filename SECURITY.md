# Security Policy
# 安全政策

This is a documentation-only Claude skill (Markdown files, no executable
code), so the attack surface is minimal. That said, if you find anything that
looks like a security issue — e.g. a prompt-injection vector in the skill
instructions, or leaked credentials/personal data anywhere in the repo or its
history — please report it privately via
[GitHub Security Advisories](../../security/advisories/new) rather than a
public issue.

這是一個純文件型的 Claude skill（只有 Markdown，沒有可執行程式碼），攻擊面
很小。但如果你發現任何疑似安全問題——例如 skill 指令中的 prompt injection
向量，或 repo 及其歷史中外洩的憑證／個資——請透過
[GitHub Security Advisories](../../security/advisories/new) 私下回報，
不要開公開 issue。
